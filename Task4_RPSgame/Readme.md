This is a command-line based **Rock, Paper, Scissors** game built using Python. The game allows the user to play multiple rounds against the computer, keeps track of scores, and displays the result after each round.

## Features
- Play against a computer that makes random choices
- User-friendly command-line interface
- Input validation for invalid entries
- Score tracking: User vs Computer
- Option to play multiple rounds
- Final score display at the end

## Instructions
- Choose:
  - `r` for Rock
  - `p` for Paper
  - `s` for Scissors
- The computer will also make its choice.
- The game determines who wins each round and keeps track of scores.

## How to Run
1. Make sure Python is installed on your system.
2. Run the script through any IDE or terminal.
